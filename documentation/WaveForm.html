Class: WaveForm

WaveForm()

The WaveForm class organizes the various parts of a WaveForm component which is the main way audio is played on Ruffmix. The component includes areas for audio playback, comments, metadata tags, title and description, and can be deleted or updated with a new version of the track, which is shown in a track 'timeline'.
